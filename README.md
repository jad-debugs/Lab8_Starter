Graceful degration and service workers are related because service workers help by caching important files so that the app will still work even if wifi connection cuts off. This is related to graceful degration because it is all about still providing a usable experience even if wifi is gone, or something else if failing. This is important because just because something like a network connection is gone, you dont want your entire app to crash, you still want it to be somewhat usable until wifi connection comes back.
